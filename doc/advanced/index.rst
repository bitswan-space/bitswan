Topics
------

**MQTT Pipeline Visibility**
    Use MQTT topics to inspect running pipelines, view topology, and
    subscribe to events flowing through components.

**Analyzers**
    Time-window analyzers for aggregation, statistics, and anomaly
    detection over streaming data.

**Custom Components**
    Build custom sources, processors, sinks, and connections for
    specialized requirements.
